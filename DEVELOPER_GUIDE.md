# DroneBloom Developer Guide

## Project Overview

DroneBloom is a single-page web application (SPA) built to showcase ongoing research integrating drones, AI, and robotics in precision agriculture.  
The site is deployed via **GitHub Pages** and accessible at [https://drone-bloom.org](https://drone-bloom.org).

## Tech Stack

-   **Frontend:** HTML, CSS, JavaScript
-   **Theme:** Matrix-inspired aesthetic
-   **Animations:** CSS-based and SVG animations
-   **Hosting:** GitHub Pages (`main` branch)
-   **Dependencies:** None (pure HTML/JS)

## Local Development
