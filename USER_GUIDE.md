# DroneBloom User Guide

## Overview

**DroneBloom** is a research project focused on integrating drone and AI technologies into orchard ecosystems to improve mapping accuracy, sustainability, and efficiency.  
The website [drone-bloom.org](https://drone-bloom.org) serves as a public landing page for sharing project updates, visual demos, and related research.

## How to Use the Site

-   Visit [https://drone-bloom.org](https://drone-bloom.org)
-   Explore the sections on project goals, research, and technology stack.
-   View demo visuals of drone simulations and robotic prototypes.
-   Access the latest version of the research paper via the “Publications” or “Docs” link.
